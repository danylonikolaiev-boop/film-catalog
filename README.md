# 🎬 MovieApp — Full-Stack Каталог Фільмів

**MovieApp** — це сучасний вебзастосунок для керування каталогом фільмів, розроблений у рамках навчальної практики. Проєкт реалізує повноцінну клієнт-серверну архітектуру (Full-Stack) та підтримує повний цикл операцій CRUD для фільмів, жанрів та відгуків.

## ✨ Основний функціонал

* **Керування фільмами:** Додавання, перегляд деталей, редагування (із автозаповненням форми) та видалення фільмів.
* **Гнучка фільтрація та пошук:** * *Серверна фільтрація:* за обраними жанрами.
  * *Клієнтська (миттєва) фільтрація:* пошук за назвою та повзунок мінімального рейтингу.
* **Система жанрів:** Окрема панель для створення та видалення категорій (жанрів).
* **Відгуки та рейтинг:** Можливість залишати коментарі з оцінкою до фільмів, видаляти їх, а також ставити "лайки" (відмітки корисності) відгукам інших користувачів.
* **Сувора валідація:** Перевірка даних форми перед відправкою на сервер (обмеження року випуску, перевірка формату посилань тощо).

## 🛠 Технологічний стек

### Frontend
* **Бібліотека:** React 18 (збірка через Vite)
* **Мова:** TypeScript
* **Маршрутизація:** React Router DOM v6
* **Керування станом/запитами:** React Query (`@tanstack/react-query`) + Axios
* **Робота з формами:** React Hook Form + Zod (схеми валідації)
* **Стилізація:** CSS Modules (ізольовані компоненти, Clean UI)

### Backend
* **Фреймворк:** Nest.js
* **Мова:** TypeScript
* **Архітектура:** Модульна (FilmsModule, GenresModule, ReviewsModule)
* **API:** RESTful API

## 🚀 Інструкція із запуску (Локально)

Оскільки проєкт складається з двох частин (клієнт та сервер), для його роботи потрібно запустити обидва середовища паралельно у двох терміналах.

### Передумови
Переконайтеся, що на вашому комп'ютері встановлено **Node.js** (рекомендовано версію 18 або вище).

### Крок 1: Запуск Backend-сервера
Відкрийте перший термінал та виконайте наступні команди:

```bash
# Перейдіть у папку бекенду
cd backend

# Встановіть залежності
npm install

# Запустіть сервер у режимі розробника
npm run start:dev
